# DMG Local Viewer

一个面向 Windows 的本地 DMG 查看与解压工具，支持未加密镜像，以及常见的 `encrcdsa` v2、AES-128 和 AES-256 加密镜像。

## 普通用户

从 GitHub Releases 下载便携包并解压，然后双击 `DMG-Local-Viewer.exe`。程序已经包含 Python 运行时、密码学组件和 7-Zip，不需要管理员权限，也不需要安装或配置 Python、Conda、7-Zip、环境变量或网络服务。

使用流程：

1. 选择 `.dmg` 文件并点击“读取文件”。
2. 程序自动识别加密状态；仅加密文件会弹出密码输入框。
3. 浏览内部文件，或点击“解压全部”。

程序不会修改原始 DMG。未加密镜像会直接以只读方式交给内置 7-Zip，不创建副本。密码不会写入配置或日志，也不会作为外部命令行参数传递。对于加密镜像，完整解密副本会暂存在系统 `%TEMP%` 中，并在程序正常退出时删除。异常退出或断电可能留下临时文件。

## 从源码构建

要求：Windows 10/11、64 位 Python 3.10 或更高版本、PowerShell 5.1 或更高版本。

双击 `build.cmd`，或在终端中运行：

```bat
build.cmd
```

脚本会自动：

1. 在项目内创建隔离的 `.venv`；
2. 安装锁定版本的 Python 依赖；
3. 从官方地址下载并校验 7-Zip 26.02；
4. 运行测试；
5. 生成自包含的 `dist\DMG-Local-Viewer.exe`；
6. 复制第三方许可证和对应的 7-Zip 源码包。

构建过程中需要网络；构建完成后的 EXE 运行时不需要网络。

## 安全边界

- 只打开可信来源的 DMG。复杂文件格式解析器始终存在潜在漏洞风险。
- 解压时程序会在用户选择的位置创建一个新的、唯一命名的子目录，避免覆盖原有文件。
- 密码是 Python 字符串，能够避免落盘，但无法保证从进程内存中进行可验证的安全擦除。
- 本项目不是 Apple 官方产品，也不受 Apple 或 7-Zip 官方支持。
